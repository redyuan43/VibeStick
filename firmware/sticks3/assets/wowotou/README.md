# Wowotou Pet Assets

These SVG files are selected source frames from the Wowotou theme in:

`MiniCPM-Desk-Pet-client/clawd-on-desk/themes/wowotou/assets`

They are used to generate small RGB565 LVGL bitmaps for the VibeStick firmware
home-screen pet. Regenerate the firmware assets with:

```sh
python3 scripts/generate_pet_assets.py
```

The user confirmed they have the right to use these Wowotou character assets in
this VibeStick repository. Do not treat these artwork files as generally
reusable project-owned MIT assets.
