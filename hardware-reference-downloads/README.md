# Hardware Reference Downloads

Offline reference bundle for VibeStick hardware costing and BOM work.

Created for:

- M5Stack StickS3
- M5StickC Plus

## Contents

- `HARDWARE_BOM.md` - local functional BOM summary copied from `docs/HARDWARE_BOM.md`.
- `StickS3/` - official StickS3 product page, schematic, model size file, STL structure, datasheets, M5PM1 page, and product/schematic images.
- `M5StickC-Plus/` - official StickC-Plus product page, schematic, STL structure, datasheets, and product/schematic images.

## Notes

- Direct official MP4 links referenced by the product pages were downloaded locally under `videos/`, but they are intentionally excluded from git because they are large. YouTube embed/player links were not downloaded because they are not direct source files.
- The downloaded schematics are useful for reference-designator extraction, but they are not a complete manufacturer production BOM.
- If a source link changes later, use the saved HTML product pages as a local snapshot of the link set used for this bundle.
